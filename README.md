# GTA6 AI Content System

This repository is my content operating system for building profitable GTA 6 AI videos for TikTok, YouTube Shorts, and Instagram Reels.

## Purpose
- Store video prompts
- Save hooks and captions
- Track content ideas
- Organize posting workflow
- Build automation later

## Folder structure
- `prompts/` = video generation prompts
- `content/` = ideas and tracker
- `captions/` = platform-specific captions
- `scripts/` = bot notes and automation docs
- `assets/` = thumbnail and creative notes

## Workflow
1. Create idea
2. Write script
3. Generate video
4. Add captions
5. Export final version
6. Track upload status
