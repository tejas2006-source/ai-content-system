# Posting Bot Notes

## Purpose

This file stores notes for future posting automation.

## Planned automation workflow

1. Read approved videos from the content tracker.
2. Match the correct caption template to the platform.
3. Upload the finished video to TikTok, YouTube Shorts, and Instagram Reels.
4. Mark the tracker row as posted.
5. Record date and status.
6. Log any posting errors for review.

## Rules

- Never post videos that are not marked as approved.
- Keep platform captions slightly different.
- Prefer scheduled posting where possible.
- Start with supervised automation before full automation.
